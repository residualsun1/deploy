5.1 简单线性回归介绍 {# OLS-inro}

在R中，拟合线性模型最基本的函数就是lm()，格式为：

myfit <- lm(formula, data)

其中，formula指要拟合的模型形式，data是一个数据框，包含了用于拟合模型的数据。

读入CGSS2021核心模块的数据。

load(file = "core.RData")

拿到一份数据，需要知道其含义。

core %>% head()
